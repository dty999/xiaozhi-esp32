# 小智AI机器人 设备状态转移图

> 此图展示 DeviceStateMachine 管理的全部合法状态转移路径

## 完整状态转移图

```mermaid
stateDiagram-v2
    [*] --> Unknown: 上电

    Unknown --> Starting: Initialize()调用

    state Starting {
        [*] --> 初始化中
        初始化中: ① 屏幕 SetupUI
        初始化中: ② 音频服务启动
        初始化中: ③ MCP工具注册
        初始化中: ④ 网络开始连接
    }

    Starting --> WifiConfiguring: 无WiFi配置<br/>进入配网模式
    Starting --> Activating: 网络连接成功<br/>启动激活流程

    state WifiConfiguring {
        [*] --> 等待配网
        等待配网: 蓝牙配网 / 声波配网
    }

    WifiConfiguring --> AudioTesting: 按键(开始音频测试)
    AudioTesting --> WifiConfiguring: 按键(停止音频测试)
    WifiConfiguring --> Activating: 网络连接成功

    state Activating {
        [*] --> 检查资源版本
        检查资源版本 --> 检查固件升级: 资源已是最新
        检查资源版本 --> 下载资源: 有新资源
        下载资源 --> 检查固件升级
        检查固件升级 --> 设备激活: 无需升级
        检查固件升级 --> Upgrading: 有新固件
        设备激活 --> 建立协议连接
        建立协议连接 --> 激活完成
    }

    Activating --> Idle: 激活完成 ✅

    state Idle {
        [*] --> 待机中
        待机中: 🔴 等待唤醒词
        待机中: ⏱️ 等待按键
        待机中: 📡 等待MCP指令
    }

    Idle --> Connecting: 唤醒词 / 按键 / MCP
    Idle --> Upgrading: 手动OTA升级指令
    Idle --> WifiConfiguring: 重新配网请求

    state Connecting {
        [*] --> 打开音频通道
        打开音频通道: 建立TLS连接
        打开音频通道: 协商音频参数
    }

    Connecting --> Listening: 通道建立成功
    Connecting --> Idle: 连接失败 / 超时

    state Listening {
        [*] --> 麦克风采集
        麦克风采集: 🎤 采集PCM音频
        麦克风采集: 🔄 OPUS实时编码
        麦克风采集: 📤 发送音频包
        麦克风采集: 🔊 等待服务器回复
    }

    Listening --> Speaking: 服务器开始TTS
    Listening --> Idle: 按键停止 / MCP停止
    Listening --> Idle: VAD静音超时

    state Speaking {
        [*] --> 播放服务器音频
        播放服务器音频: 📥 接收OPUS音频包
        播放服务器音频: 🔄 OPUS解码
        播放服务器音频: 🔊 扬声器播放
    }

    Speaking --> Listening: TTS结束(AutoStop模式)<br/>自动继续收音
    Speaking --> Idle: TTS结束(ManualStop模式)<br/>对话完成
    Speaking --> Listening: 唤醒词打断<br/>用户想重新说话

    state Upgrading {
        [*] --> 下载固件
        下载固件 --> 刷写固件
        刷写固件 --> 重启: 升级成功
        下载固件 --> Idle: 升级失败
    }

    Speaking --> Idle: 按键中止
    Listening --> Listening: VAD检测到人声
```

## 各状态详解

| 状态 | 枚举值 | 描述 | 屏幕显示 |
|------|--------|------|----------|
| **Unknown** | `kDeviceStateUnknown` | 启动前默认状态 | - |
| **Starting** | `kDeviceStateStarting` | 系统初始化中 | "启动中..." |
| **WifiConfiguring** | `kDeviceStateWifiConfiguring` | Wi-Fi 配网模式 | "等待配网" |
| **AudioTesting** | `kDeviceStateAudioTesting` | 音频硬件测试 | "音频测试" |
| **Activating** | `kDeviceStateActivating` | 激活流程中（升级+鉴权） | "检查更新..." |
| **Idle** | `kDeviceStateIdle` | 待机就绪，等唤醒词 | "就绪" |
| **Connecting** | `kDeviceStateConnecting` | 正在打开音频通道 | "连接中..." |
| **Listening** | `kDeviceStateListening` | 正在收音（对话中） | "正在听..." |
| **Speaking** | `kDeviceStateSpeaking` | 正在播放AI回复 | "正在说..." |
| **Upgrading** | `kDeviceStateUpgrading` | 固件 OTA 升级中 | "升级中..." |

## 关键限制规则

| 规则 | 说明 |
|------|------|
| `Idle → Listening` 直接跳转 | **不允许**，必须先经过 `Connecting`（打开音频通道） |
| `Speaking → Listening` 自动跳转 | 仅在 AutoStop 模式下允许（AEC 关闭时默认） |
| `Listening → Speaking` 不被允许 | 状态切换由服务器 TTS 消息驱动，非设备主动 |
| 任何状态都可以 → `Upgrading` | 固件升级优先级最高 |
| `Upgrading → Idle` 仅失败时 | 升级成功则直接 `Reboot()`，不回到 Idle |
