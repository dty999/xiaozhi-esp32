# 小智AI机器人 启动时序图

> 此图展示设备从上电复位到进入主事件循环的完整启动时序

## 启动流程（上电 → 主循环）

```mermaid
sequenceDiagram
    participant PWR as 上电/复位
    participant ROM as Boot ROM
    participant BL as Bootloader
    participant MAIN as main.cc<br/>app_main()
    participant APP as Application<br/>单例总管
    participant BOARD as Board<br/>硬件总管
    participant AUDIO as AudioService<br/>音频服务
    participant MCP as McpServer<br/>MCP工具箱
    participant NET as 网络模块<br/>WiFi/4G
    participant EVT as 事件组<br/>13铃总线
    participant LOOP as Run()<br/>主事件循环
    participant SRV as 云端服务器<br/>xiaozhi.me

    Note over PWR,SRV: ═══ 第一阶段：硬件启动（固件前置流程）═══
    PWR->>ROM: 芯片上电 / 复位引脚触发
    ROM->>BL: 加载 Bootloader（第2阶段引导）
    BL->>MAIN: 加载 app_main()（用户程序入口）

    Note over MAIN,SRV: ═══ 第二阶段：app_main() 初始化 ═══
    MAIN->>MAIN: nvs_flash_init()<br/>初始化NVS存储（读WiFi密码等）
    MAIN->>APP: Application::GetInstance()
    activate APP
    APP->>EVT: xEventGroupCreate()<br/>创建事件组（13铃总线）
    Note right of EVT: 一劳永逸

    MAIN->>APP: Initialize()

    Note over APP,NET: ═══ 第三阶段：Initialize() 各模块初始化 ═══
    APP->>BOARD: GetDisplay().SetupUI()<br/>初始化显示屏UI
    APP->>BOARD: GetAudioCodec()
    APP->>AUDIO: Initialize(codec)<br/>初始化编解码器与麦克风
    APP->>AUDIO: Start()<br/>启动音频采集线程
    APP->>AUDIO: SetCallbacks(...)<br/>注册3个音频回调
    Note right of AUDIO: 铃1:音频就绪<br/>铃2:唤醒词<br/>铃3:VAD变化

    APP->>EVT: AddStateChangeListener()<br/>注册状态变化回调
    Note right of EVT: 铃12:状态变化

    APP->>EVT: esp_timer_start_periodic()<br/>启动1秒时钟定时器
    Note right of EVT: 铃6:每秒响

    APP->>MCP: GetInstance().AddCommonTools()<br/>注册通用工具
    Note right of MCP: get_battery_level<br/>volume.set/get<br/>reboot 等

    APP->>MCP: AddUserOnlyTools()<br/>注册用户专属工具

    APP->>NET: SetNetworkEventCallback(...)<br/>注册网络回调
    Note right of NET: 铃7:网络连接<br/>铃8:网络断开

    APP->>NET: StartNetwork()<br/>异步启动网络连接
    Note right of NET: 后台任务不阻塞

    APP-->>MAIN: Initialize() 返回

    Note over MAIN,SRV: ═══ 第四阶段：进入主循环 ═══
    MAIN->>APP: Run()
    APP->>EVT: vTaskPrioritySet(10)<br/>提升主任务优先级
    Note right of EVT: 确保音频数据<br/>优先被处理

    loop 永恒循环（直至断电）
        APP->>EVT: xEventGroupWaitBits()<br/>阻塞等待任一铃响
        Note right of EVT: 13种事件任意一个触发
        
        alt 铃7：网络已连接
            EVT-->>APP: NETWORK_CONNECTED
            APP->>NET: HandleNetworkConnectedEvent()
            APP->>APP: 创建激活后台任务
            Note right of APP: 激活任务：<br/>①CheckAssetsVersion<br/>②CheckNewVersion<br/>③InitializeProtocol
        end

        alt 激活任务完成
            APP->>SRV: 建立通信协议连接
            SRV-->>APP: 返回配置信息
            APP->>EVT: 铃5 ACTION_DONE
            APP->>APP: HandleActivationDoneEvent()
            Note right of APP: 设备就绪<br/>等待"小智小智"
        end

        alt 铃2：唤醒词检测
            EVT-->>APP: WAKE_WORD_DETECTED
            APP->>SRV: 打开音频通道
            APP->>SRV: 发送唤醒词音频
            APP->>APP: 进入 Listening 状态
        end
    end
```

## 关键时间节点

| 时间点 | 事件 | 对应代码 |
|--------|------|----------|
| T+0s | 芯片上电 | Boot ROM 自动执行 |
| T+0.1s | Bootloader 启动 | 第二阶段引导 |
| T+0.3s | `app_main()` 入口 | `main.cc:14` |
| T+0.4s | 事件组、时钟创建 | `Application::Application()` |
| T+0.5s | 屏幕、音频、MCP初始化 | `Application::Initialize()` |
| T+0.6s | 网络异步连接开始 | `board.StartNetwork()` |
| T+0.7s | 进入主循环 `Run()` | `application.cc:165` |
| T+1~10s | 网络连接成功 | `HandleNetworkConnectedEvent()` |
| T+1~60s | 激活完成 | `HandleActivationDoneEvent()` |
| T+1s+ | 设备就绪，等待唤醒词 | Idle 状态 |
